Quiet signals, everywhere you look.

Status bar, command palette, notifications, settings: the four smallest surfaces the extension touches. Each carries the minimum useful signal and routes back to a full view in one click.

Implementation notes

  • Status bar = three window.createStatusBarItem entries (state · metrics · gate). State + heartbeat poll .zo/runtime/session.json every 2s; the metrics segment tails .zo/experiments/exp-003/metrics.jsonl and training_status.json; the gate item flips on gate_pending events in logs/comms/2026-07-12.jsonl. Clicks: state → attach terminal, gate → gate review editor.
  • Palette commands map 1:1 onto the CLI: Start Run → zo build; Attach to Session → zo continue --headless-tmux; Approve Gate → /gates:approve (Reject via the review panel → /gates:reject); Set Gate Mode → zo gates set; Send Message to Lead → zo runtime send; Stop Session → runtime teardown of tmux session zo-mnist-20260712-091412. The two Open commands reveal webview panels and invoke nothing.
  • Approve Gate's detail line is composed from the latest oracle event in the comms feed (test_accuracy 0.9912 vs tiers 0.95 / 0.98 / 0.99 → could_pass); the command is hidden behind a zo.gatePending context key when no gate is open.
  • Toasts fire from the comms JSONL tailer: gate_pending → warning toast with actions, run_complete → info toast with Open summary, api_backoff → passive toast, auto-dismissed on recovery. One toast per event type per phase; everything else is status bar only, governed by zo.notifications.gates.
  • zo.appearance = brand renders webviews with these tokens; editor swaps each token for its --vscode-* fallback per the mapping table in the tokens.css production note. Status bar and toasts always use native chrome regardless of this setting.
  • zo.pythonPath auto-detect runs uv python find on activation; a manual override is validated with an interpreter probe before first use.
  • Keybindings ship in package.json under the ⌘⇧Z chord namespace. Deferred from v1: chord remapping UI, per-project notification levels, and a quick-pick submenu for Set Gate Mode (v1 opens a plain three-option pick).
